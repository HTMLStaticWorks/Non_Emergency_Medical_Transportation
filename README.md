# NEMT Care Transportation Website

A premium service-based and booking platform for Non-Emergency Medical Transportation.

## Features
- Multi-step Ride Booking
- Live Vehicle Tracking UI
- Admin Dashboard
- Patient & Facility Portals
- Responsive 1100px Breakpoint
- Dark Mode Support

## Tech Stack
- Bootstrap 5
- Bootstrap Icons
- Vanilla JavaScript
- Custom CSS Variables System

## Installation
Open `index.html` in any modern web browser.
